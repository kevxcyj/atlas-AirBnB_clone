# AirBnB Clone - The Console

## Synopsis

HBNB is the console that is going to be used for the AirBnB Clone

## Description

This is the console, known as HBNB, which is used to create, change, delete and remove objects in different classes. It is built on python3 and will use serialization to create the file.json file. This file is needed for certain commands to function.

## Commands
| **Command** | **Function** |
| ----------- | ------------ |
| create | Creates a new instance |
| show | Prints the string representation |
| destroy | Deletes |
| quit | Exits the console |
| EOF | Exits the console |
| help | Shows commands |

## Usage

Once entering the consoole, the prompt will appear as 

(hbnb)

You can then use the commands listed above to complete certain tasks.

EX:

(hbnb)create BaseModel
760c35ef-9a1e-428b-8a44-ad1f3683cc78

Creates a new instance of BaseModel using the create command.

(hbnb)help

Documented commands (type help <topic>):
EOF  help  quit

Lists commands that are documented.
